# Stratify – Responsive Consultancy Website

The Stratify is a simple, completely responsive website of a consultancy constructed with pure HTML and CSS.
This project is aimed at a structured layout, responsive web principles, and clean UI implementation without any frameworks or JavaScript.

---

## Overview

 Stratify is a modern site of consulting firm that includes:

- Home page with hero, highlights, and call-to-action sections  
- Services page consisting of structured service cards.  
- About page with a minimal two-section layout  
- Contact page with a clean and professional form  
- Fully responsive design using CSS Grid and Flexbox  

The design is based on a neutral and executive color scheme and a uniform typography system.

---

## Project Goals

- Practice Responsive Web Design (RWD)
- Implement CSS Grid and Flexbox effectively
- Maintain clean and scalable CSS architecture
- Build a professional multi-page layout
- Develop without external libraries or frameworks
